# Screen_And_Translate (В РАЗРАБОТКЕ)

Скрипт, который из скриншота слова делает распознавание слова, перевод слова с помощью Google API и сохраняет
результаты в два файла. Один текстовый файл содержит искомое слово и его перевод, а второй json файл хранит перевод,
а также определение слова.

Если перевод слова на другом языке, то в файл записывается только искомое слово и его перевод без определения

# Запуск проекта:

Клонировать репозиторий и перейти в него в терминале:
```bash
git clone https://github.com/NGC6543/Screen_And_Translate.git
```

```bash
cd Screenshot_Word
```

Установить зависимости:

```bash
pip install -r requirements.txt
```

Запустить главный файл:
```bash
py screen_and_translate.py
```
